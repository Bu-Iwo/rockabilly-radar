# Rockabilly Radar Event Scraper

Dieses System scrapt automatisch Events von verschiedenen Websites und aktualisiert die `events.json` Datei.

## Quellen

- **we-love-country.de**: 670+ Country & Line Dance Events
- **swingcalendar.com**: 234+ internationale Swing Events
- **swingindd.com**: 140+ Dresden Swing Events

## Installation

### Lokal testen

```bash
# Dependencies installieren
pip install -r requirements.txt

# Scraper ausführen
python scraper.py
